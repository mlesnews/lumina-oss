# LUMINA Unified Queue Status Bar Extension

Displays the unified queue (sources, problems, tasks) status in the IDE footer/status bar.

## Features

- **Real-time Status**: Shows queue summary in footer
- **Visual Indicators**: Icons for processing, pending, failed items
- **Click for Details**: Click status bar item to see full queue details
- **Auto-updates**: Refreshes every 2 seconds
- **File Watcher**: Updates immediately when queue state changes

## Status Bar Display

The status bar shows:
- Total queue items
- Processing items (with spinner)
- Pending items
- Failed items
- Completed items (if queue is small)

Example: `📋 ⚙️ 15 ⏳3 ❌1` means:
- 15 total items
- Currently processing
- 3 pending
- 1 failed

## Commands

- **Show Queue Details**: Click status bar item or run command
- **Open Queue Viewer**: Opens terminal with full queue viewer
- **Refresh Queue Status**: Manually refresh status

## Configuration

```json
{
  "lumina.unifiedQueue.showInStatusBar": true,
  "lumina.unifiedQueue.updateInterval": 2000,
  "lumina.unifiedQueue.showDetails": true
}
```

## Installation

1. Navigate to extension directory
2. Run `npm install`
3. Run `npm run compile`
4. Press F5 to launch extension development host
5. Or package and install: `vsce package` then install `.vsix` file

## Requirements

- Queue state file at: `data/unified_queue/queue_state.json`
- Generated by: `unified_queue_adapter.py`
